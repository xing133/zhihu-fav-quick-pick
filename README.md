# zhihu-fav-quick-pick

油猴脚本：知乎收藏夹弹窗中，加了一个过滤器，可以快速选中。

吐槽一下知乎的糟糕设计：收藏夹的顺序似乎是“上次选中后向上移动排位”的，但又似乎不是，
然后列表还是动态加载的，然后加载中顺序好像还会乱跳。。
结果就是没法记位置，找又找不到，每次收藏都一脑子门子邪火。。
